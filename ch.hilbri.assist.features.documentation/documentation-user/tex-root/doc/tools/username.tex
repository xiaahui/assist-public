% Usage
\section{Usage}
Simply load the package via \lstinline!\usepackage{username}!. It will then provide the following commands.

\begin{itemize}
	\item \lstinline!\userdomain! holding the current user's domain:\\
	      \userdomain
	\item \lstinline!\ifuserdomain!  to test, if user domain was correctly read.
	\item \lstinline!\username! holding the current username:\\
	      \username
	\item \lstinline!\ifusername!  to test, if username was correctly read.
\end{itemize}
